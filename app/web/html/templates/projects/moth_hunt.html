Moth Hunt

Sorry this game doesn't work on mobile browsers

Batty's Moth Hunt was the first game and one of the first programs I ever wrote (back in 2017). It's written in Scratch, a block-style, learner programming language created by MIT. If you have someone interested in programming, I highly recommend pointing them to this language as a first try. It was a fun experience that excited me to learn to program with "real" programming languages.

In this desktop browser-based game, you play as a bat. You aim to hunt down moths for dinner using echolocation to find your prey. While hunting, you must avoid the owl hunting you. Aim your mouse where you'd like Batty to travel and press space bar to use his echolocation powers to reveal moths and the owl in the night sky.
